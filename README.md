# Weather Forecast Dashboard

A responsive weather forecast web application built using **HTML, Tailwind CSS, and Vanilla JavaScript**.  
The application fetches real-time weather data from the **OpenWeatherMap API** and displays current weather conditions along with a 5-day forecast.
